Cloud Solution JOB DESCRITION
Position: Cloud Solutions Engineer
Responsibilities:
- Design, implement, and manage cloud infrastructure on AWS, Azure, or GCP.
- Automate deployments using Terraform or CloudFormation.
- Ensure scalability, availability, and security of applications.
- Work with Kubernetes, Docker, and CI/CD tools.
Required Skills:
- AWS or Azure experience
- Terraform, Docker, Jenkins, Python
- Monitoring tools like CloudWatch, Prometheus
- Strong understanding of cloud security
Preferred:
- AWS/Azure certification
- Experience with GCP is a bonus
